# Lista de Tarefas

Um aplicativo simples de lista de tarefas usando PyQt6 e MySQL.

## Pré-requisitos

- Python 3.7 ou superior
- PyQt6
- mysql-connector-python

## Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/ThiagoGterceiro/Sistema-CRUD-de-Tarefas-Python.git

2. Acesse o diretório do projeto:
  cd lista-de-tarefas

3. Crie um ambiente virtual (opcional):
  python -m venv venv
  source venv/bin/activate

4. Instale as dependências:
  pip install -r requirements.txt

5. 
Configuração do banco de dados:

Certifique-se de ter o MySQL instalado e em execução.
Crie um novo banco de dados no MySQL com o nome "lista_de_tarefas".
No arquivo Controle.py, atualize as credenciais de conexão ao banco de dados (user, password, host) conforme necessário.


Funcionalidades

Adicionar tarefas com título, descrição, data de criação e status.
Listar todas as tarefas cadastradas.
Editar tarefas existentes.
Excluir tarefas.
Mudar o status de uma tarefa entre "Progresso" e "Concluído".
